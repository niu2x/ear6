# Ear6 Web 界面设计

## 设计方向：90 年代消费电子

模拟器本质上是把旧硬件请进浏览器。界面设计围绕 **90 年代 Hi-Fi / CRT 家电** 的美学展开——暗色拉丝金属机身、荧光指示灯、物理按钮质感的控件、略带弧度的 CRT 屏幕。每个模拟系统像是一台独立设备，切换标签等于切换"信号源"。

整体调性：**克制但温暖，工业但精致**。不是复古像素风，而是复古硬件风。

---

## 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 机身色 | 深炭灰 `#1a1a1e` | 主背景，模拟拉丝金属 |
| 面板色 | 深灰 `#242429` | 控制面板、标签栏底色 |
| 屏幕边框 | 纯黑 `#0a0a0a` | 画面区外框，模拟 CRT 边框 |
| 主色 — NES | 暖红 `#e63946` | 指示灯、激活态、NES 标签 |
| 主色 — Test | 示波绿 `#2ec4b6` | Test 标签及对应控件 |
| 主色 — Flash | 电光蓝 `#4ea8de` | Flash 标签及对应控件 |
| 荧光线 | 暖白 `#f1faee` | 高亮文字、开启状态 |
| 文字 | 冷灰 `#a0a0b0` | 次级信息、标签未选中 |

每个系统标签激活时，控制栏的 accent 色切换为对应的系统主色。

```
│  ⬤ Ear6                          ▓ ▒ ░     │

│   NES│  Test│  Flash│      ← 信号源选择器    │
├──────┴──────┴───────┴────────────────────────┤
│ ┌──────────────────────────────────────────┐ │
│ │                                          │ │
│ │        ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐           │ │
│ │        │                   │             │ │
│ │        │    画面区域        │  ← CRT 屏幕 │ │
│ │        │    (保持比例)      │             │ │
│ │        │                   │             │ │
│ │        └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘           │ │
│ │                                          │ │

│  就绪                                60 FPS  │
└─────────────────────────────────────────────┘
```

### 标题栏

最窄的一行，仅容纳：
- 左侧：一个小圆点指示灯 + "Ear6" 字样。指示灯颜色反映当前状态（灰=空闲，绿=运行，黄=暂停）。
- 右侧：三个极小的圆点装饰（无功能，纯氛围）。

### 信号源选择器（标签栏）

三个标签并排，无间距，底部有一条细线指示当前选中。标签样式模拟老式功放机的输入选择按钮——方形，文字居中，选中时标签下方亮起一条 accent 色细线，且文字加粗。

标签切换时，下方控制面板的 accent 色渐变为目标系统色。

### CRT 画面区

画面区是整个页面的视觉重心，占据约 70% 的垂直空间。

- 背景纯黑，模拟 CRT 关机状态
- 未加载 ROM 时，画面区中央显示一段风格化的待机文字（细等宽字体，缓慢呼吸闪烁）
- 加载 ROM 后，画面占据画面区，保持原始像素比例，居中
- 画面区外框有极细微的内阴影，模拟 CRT 屏幕凹陷感
- 四角有轻微的圆弧

### 控制栏

控制栏位于画面区下方，所有控件左对齐，ROM 文件名右对齐。

控件样式：扁平但有微妙的按压感——深色底、窄边框、hover 时背景提亮、active 时下沉 1px。主要操作（运行/暂停）使用当前系统的 accent 色。

| 控件 | 说明 |
|------|------|
| ⏻ 打开 | 触发文件选择。加载成功后按钮右侧出现文件名 |
| ↺ 重置 | 重新加载当前 ROM，回到暂停 |
| ▶ / ⏸ 运行/暂停 | 切换运行状态。无 ROM 时禁用（灰色，不可点击） |
| 区域 [▼] | NES 专用：NTSC / PAL 切换 |

### 状态指示灯

控制在运行时在标题栏 NES 标签右侧显示三个极小的圆点状态灯——左=电源（常亮），中=运行（闪烁），右=错误（仅在出错时亮起）。

### 状态栏

最底部一行，极细字体：
- 左侧：状态文字（就绪 / 加载中 / 运行中 / 暂停 / 已重置）
- 右侧：实时 FPS 计数器

## 交互细节

### 切换系统

切换标签时，画面区先渐黑（200ms），然后新系统控制面板渐入（accent 色同时过渡），画面区恢复显示。如果新系统之前加载过 ROM，恢复最后一帧；否则显示待机文字。

### 播放/暂停

- 从暂停到播放：画面区短促的"唤醒"动画——轻微亮度脉冲（模拟 CRT 开机）
- 从播放到暂停：画面定格，无过渡

### 重置

按下重置：画面区闪烁一次（50ms 全白 → 恢复），模拟 CRT 关机/开机效果。

### 加载 ROM

文件选择后，状态栏显示"加载中..."，加载完成后画面显示第一帧，状态栏恢复"就绪"。

## 各系统面板

### NES

完整的控制面板：打开 ROM、重置、运行/暂停、区域选择。键盘映射支持方向键 + Z/X + Enter/Shift。

### Test

控制面板简化为一个"▶ 运行测试"按钮，accent 色为绿色。无其他选项。

### Flash

面板显示居中文字"Flash 模拟器 · 即将推出"，所有控件隐藏。标签切换行为与其他系统一致，但画面区始终显示待机画面。

## 后续可扩展

- **扫描线覆盖**：可选开启 CRT 扫描线效果
- **画面模式**：原始比例 / 4:3 拉伸 / 全屏
- **音频**：音量滑块 + 静音按钮
- **存档**：多槽位保存 / 读取，每个槽位显示时间戳
- **按键映射**：可点击修改的键盘绑定表
- **拖拽加载**：拖拽 ROM 到画面区松开即加载
- **触屏**：移动端半透明虚拟按键覆盖层
- **金手指**：作弊码输入 + 启用列表
